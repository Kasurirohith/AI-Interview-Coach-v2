# AI Interview Coach

AI-powered interview preparation platform built using React, FastAPI, and SQLite.

## Features
- Resume ATS Score Analysis
- Resume Skill Extraction
- AI Generated Interview Questions
- Voice-Based Mock Interview
- Speech Recognition
- Interview Timer
- Dashboard Analytics
- Interview History Tracking
- Dark/Light Mode
- User Authentication

## Tech Stack
Frontend:
- React.js
- Vite
- Recharts
- CSS

Backend:
- FastAPI
- Python
- SQLite
